Netflix Movies Data Analysis (1990s Focus)

Data analysis project exploring Netflix movie datasets with a focus on 1990s trends, duration distribution, and short movie insights.

---

 Project Overview

This project explores Netflix movies from the 1990s decade (1990–1999) to understand patterns in:

- Movie duration trends

- Most common movie lengths

- Short movies (less than 90 minutes)
